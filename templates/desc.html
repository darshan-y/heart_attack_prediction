Description

        1. age

        2. sex:
            0 = Female
            1 = Male

        3. cp: chest pain type
            Value 1: typical angina
            Value 2: atypical angina
            Value 3: non-anginal pain
            Value 4: asymptomatic

        4. resting blood pressure
            trestbps: resting blood pressure (in mm Hg on admission to the hospital)

        5. serum cholestoral in mg/dl
            chol: serum cholestoral in mg/dl

        6. resting electrocardiographic results (values 0,1,2)
            restecg: resting electrocardiographic results
            Value 0: normal
            Value 1: having ST-T wave abnormality (T wave inversions and/or ST elevation or depression of > 0.05 mV)
            Value 2: showing probable or definite left ventricular hypertrophy by Estes criteria

        7. oldpeak = ST depression induced by exercise relative to rest

        8. maximum heart rate achieved

        9. exercise induced angina
            exang: exercise induced angina (1 = yes; 0 = no)

       10. the slope of the peak exercise ST segment
            slope: the slope of the peak exercise ST segment
            Value 1: upsloping
            Value 2: flat
            Value 3: downsloping

       11. number of major vessels (0-3) colored by flourosopy
            thal: 1 = normal; 
            thal:  2 = fixed defect; 
            thal:  3 = reversable defect
            Thalassemia (thal-uh-SEE-me-uh) is an inherited blood disorder that causes your body to have less hemoglobin than normal. Hemoglobin enables red blood cells to carry oxygen. Thalassemia can cause anemia, leaving you fatigued. If you have mild thalassemia, you might not need treatment.
